📱 PWA Installation
📱 This app can be installed on your phone as a native app!
Android (Chrome): Tap ⋮ menu → "Add to Home Screen"
iPhone (Safari): Tap Share icon → "Add to Home Screen"
Desktop Chrome: Click install icon in address bar
💾 Data Management
📖 Google Apps Script Setup Guide
Step 1: Open your Google Sheet → Extensions → Apps Script
Step 2: Paste the code from Code.gs (included in package)
Step 3: Deploy → New Deployment → Web App → Anyone can access
Step 4: Copy the Web App URL and paste above
Step 5: Click "Connect & Test" to verify connection
